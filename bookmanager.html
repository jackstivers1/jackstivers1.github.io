Project Overview

Book Manager is my solo project focused on creating a practical full-stack web application with a deployed frontend, backend logic, and persistent database storage.

What I Built

  • Created a user-facing book management interface.
  • Connected the app to PostgreSQL-backed book data.
  • Set up deployment so the project can be opened and tested live.
  • Connected the app to a custom domain for a more professional presentation.
